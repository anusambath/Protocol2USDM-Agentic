Hover State Standards

  • ✓ Transition Duration: 150ms for all interactions
  • ✓ Easing Function: ease-in-out for smooth, natural feel
  • ✓ Visual Feedback: Background color change, text color change, or shadow
  • ✓ Focus Integration: Focus ring visible with hover state
  • ✓ Dark Mode: All colors adapt using semantic tokens
  • ✓ Disabled States: No hover effect, reduced opacity
