# Keyboard Shortcuts Reference

Quick reference guide for keyboard shortcuts in the provenance UI.

## Global Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Esc` | Close sidebar | When sidebar is open and unpinned |
| `Cmd+K` / `Ctrl+K` | Open search | Anywhere (future implementation) |
| `Cmd+P` / `Ctrl+P` | Open preview | Anywhere (future implementation) |

## Protocol Preview Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `←` (Left Arrow) | Previous page | When viewing protocol pages |
| `→` (Right Arrow) | Next page | When viewing protocol pages |
| `+` or `=` | Zoom in | When viewing protocol pages |
| `-` | Zoom out | When viewing protocol pages |

## Split Pane Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `↑` (Up Arrow) | Decrease top pane size | When split divider is focused |
| `↓` (Down Arrow) | Increase top pane size | When split divider is focused |

## Navigation Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Tab` | Move to next element | Anywhere |
| `Shift+Tab` | Move to previous element | Anywhere |
| `Enter` | Activate button/link | When button/link is focused |
| `Space` | Activate button | When button is focused |

## Tips

- Shortcuts are disabled when typing in input fields
- Use `Tab` to navigate between interactive elements
- Focus indicators (blue ring) show which element is active
- Press `Esc` to quickly close the sidebar
- Arrow keys work in protocol preview for quick page navigation
- Zoom shortcuts work without modifier keys for quick access

## Accessibility Features

- All interactive elements are keyboard accessible
- Visible focus indicators on all elements
- Screen reader support with ARIA labels
- Skip links for quick navigation
- Logical tab order throughout the UI

## Browser Compatibility

- Chrome/Edge: Full support
- Firefox: Full support
- Safari: Full support
- Note: Some shortcuts may conflict with browser defaults (e.g., Cmd+P for print)
